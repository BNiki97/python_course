Az Anaconda python disztribúció telepítése és használata

Az Anaconda

A Python-t többféleképpen is fel lehet telpíteni. Linux alapú rendszereken jó eséllyel már fel van telepítve, így gond nélkül használható.

Gyakorlatban nem a Python parancsértelmező telepítése, hanem a Python csomagok kezelése a nagyobb kihívás. Szerencsére a probléma megoldására már több programot is kifejlesztettek. Az egyik legelterjedtebb az Anaconda python disztribúció.

Az Anaconda nem csak a python parancsértelmezőt tartalmaz, hanem a egy csomagkezelőt, grafikus kezelőfelületet és további hasznos programokat.

A telepítés lépései

  1. Keressük fel az Anaconda honlapját; link. A Downloads menüpontra kattintva a webplap átirányít a letöltésekhez. Itt kétfajta disztribúciót tölthetünk le:
    • Az egyik egy régebbi Python (v 2.7) verzióra alapul. Ezt a verziót nem ajánlom, csak régebbi csmagokkal való kompatibilitás érdekében érhető el.
    • A másik a legutolsó stabil Python verzión alapul (v 3.*). A legtöbb csomag ezzel kompatibilis, ennek a használatát ajánlom.

  1. A letöltött .exe fájlt futtatva a program telepíthető.

  2. A telepítés után több program is rendelkezésünkre áll.
    • Az Anaconda Navigator a grafikus felhasználói felületet indítja el.
    • Az Anaconda Powershell Prompt pedig a parancssoros alkalmazást, azoknak akiknek a parancssor jobban kézre áll.
    • A Jupyter Notebook egy web alkalmazást indít el. A notebook formátum segítségével interaktívan lehet Python scripteket készíteni. Erről a következő dokumentumban lesz szó.

A grafikus alkalmazás

Az Anaconda Navigator-t elindítva a következő képernyő fogad minket.

Elérhető programok

Itt további alkalmazások érhetőek el. A Jupyter Notebook-ot már említettem. A JupyterLab tulajdonképpen a következő generációja a Notebook programnak, funkcionalítás szempontjából nagyon hasonló.

A Spyder egy Matlab-hoz hasonló grafikus felületet biztosít a Python scriptek írásához. Számtalan funkciója közül a két talán leghasznosabb

  • a dokumentáció-böngészőt, mely a különböző függvények dokumentációját teszi elérhetővé,
  • definált változók értékeinek megtekintése, böngészése (pl. mátrixok értékeinek áttekintése).

A csomagkezelő

A beépített csomagkezelő, Environments menüpont, képes különböző python csomagok telepítésére, csomagok verzióinak kezelésére.

Az Anaconda disztribúció alapból tartalmaz jónéhány beépített csomagot.

Ha a letöltött disztribúció mégsem tartalmazná a numpy, scipy, matplotlib csomagokat telepítsük ezeket.

  • A numpy csomag n-dimenzionális tenzorok kezelését (vektor, mátrix) és az ezekhez tartozó aritmetikai és lineáris algebrai műveleteket teszi elérhetővé (pl. vektorok skalárszorzata, mátrixszorzat, egyenletrendszer megoldása, mátrix invertálása, FFT - gyors Fourier-transzformáció).
  • A scipy csomag a numpy funkcionalitását bővíti ki további függvényekkel: jelfeldolgozási függvények, interpoláció, statisztikák, ritka mátrixok, Matlab adatfájlok (*.mat) írása és olvasása, speciális függvények.
  • A matplotlib csomag Matlab parancsokhoz hasonló ábrakészítő függvényeket tartalmaz.

A fentebb felsorolt csomagok maguk is rengeteg függvényt tartalmaznak, melyeket nem fogunk tudni mind érinteni a kurzus során, de dokumentációjuk elérhető az interneten.

A csomagok használatával a különböző notebook-okban fogunk megismerkedni.

A Jupyter Notebook

A kurzus során a notebook formátumot fogjuk használni a legtöbbet. A notebook fájl (*.ipynb), egy json formátú szövegfájl, melyet a Jupyter Notebook alkalmazással tudunk megnyitni. Magáról a notebook-ról és funkcionalitásáról a következő jegyzetben lesz szó.

Az Anaconda Navigator-ból elindítva a Jupyter Notebook-ot, az alkalmazás az alapértelmezett böngésző pogramban egy új lapot megnyitva jelenik meg.

Az alkalmazást a böngészőből tudjuk kezelni. Tekintsük át a legfontosabb funkciókat:

  • A mappanevekre kattintva beléphetünk az adott mappába, hasonlóan egy fáljkezelő programhoz.
  • Egy adott fájlra kattintva a program megnyitja azt. Egyszerű szövegfájl esetén, egy szövegszerkesztő programot indít el. Képfájlokat új böngésző lapon nyit meg. Notebook fájlra kattintva megnyitja azt egy új böngésző lapon.
  • A New menüpont alatt létrehozhatunk új mappát,szöveges fájlt és notebook-ot.
  • A Quit gomb, értelemszerűen leállítja az alkalmazást.
