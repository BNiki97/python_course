Jegyzetek

A kurzushoz készített notebook-ok a github.com/bozso/python_course címen érhetőek el. Innen letöltve a Jupyter Notebook alkalmazással lehet őket megnyitni, szerkeszteni és futtatni.

A slack szoba címe.

A notebook fájlokat, többek között, html formátumba lehet exportálni. Az exportált notebook-ok a tartalomjegyzék linkjeire kattintva érhetőek el.

Tartalomjegyzék

  1. Bevezetés a Python programozási nyelvbe.
  2. Az Anaconda disztribúcó telepítése.
  3. Bevezetés a Python értelmezőbe a notebook formátumba és Jupyter Notebook használatába.
  4. Bemenet és kimenet kezelése, fájlok írása olvasása, hibakezelés.
  5. Bevezetés a numpy csomag használatába.
  6. Lineáris algebrai függvények a numpy-ban, eredmények vizualizációja (matplotlib).
